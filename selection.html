Filter Chips

Single or multi-select filtering. Click to toggle active state.

When to use / avoid
When to use
  • Multi-category filtering on lists or grids.
Avoid when
  • Binary toggles — use a switch instead.

Input Chips (Removable)

Tag-style input for multi-value fields. Press Enter to add, click × to remove.

Tags

Press Enter to add a tag. Click × to remove.

When to use / avoid
When to use
  • Multi-value text inputs (tags, recipients).
Avoid when
  • Values should come from a predefined list — use multiselect picklist.

When to use
  • Single or multi-file selection with visual feedback.
  • Drag-and-drop from desktop or file manager.
Avoid when
  • Only a single file is needed with no preview — use a plain file input.
  • Large batch imports — use a dedicated import flow with progress.
