Patina is a design language for interfaces that deserve to age gracefully. Built on the conviction that warmth is not the enemy of professionalism, it trades the sterile blues and sharp corners of enterprise monotony for earthen tones, generous typography, and surfaces that feel like they were made by someone who has actually used a library card.

Every token, every radius, every shadow has been chosen with the quiet confidence of a typeface that doesn't need to shout. If your product aspires to be more than a spreadsheet with rounded corners, you've come to the right place.

Color

Four chromatic families, each with a job. Rust is the primary voice—assertive without being aggressive. Lichen provides organic calm. Chicory brings clarity when data needs to speak. And Amethyst marks the boundary where human authorship ends and machine intelligence begins.

Typography

Three type families, each with a clear mandate. Display type has presence. Body type has legibility. Monospace type has precision. There is no fourth family because three is already one more than most systems use well.

Philosophy

Most design systems begin with a color picker and end with a Jira ticket. Patina begins with a question: what if software looked like it was designed by people who read books printed on paper? The answer, it turns out, is a system built on restraint, warmth, and an unfashionable respect for the written word.

We use serif headings not because we're nostalgic, but because display type should have a voice. We chose a rust accent not because it trends on Dribbble, but because terracotta has been beautiful for four thousand years and will remain so long after your current primary blue has been deprecated.

Principles

Warm over clinical

Backgrounds that breathe. Borders that recede. Color that feels like it came from the earth, not a hex generator.

Typographic hierarchy

Four tiers of text color. Three font families, each with a purpose. Display type in Noto Serif, because headings should have presence.

Quiet surfaces

Subtle grain texture. Organic background shapes. Shadows that suggest depth without demanding attention.

Machine-aware

AI-generated content is visually distinct. The amethyst palette signals where human authorship ends and machine intelligence begins.

Restraint as a feature

Seven spacing values. Six border radii. Five shadow levels. Everything you need and nothing you don't.

Considered motion

Spring easings on toggles. Staggered card entrances. Animation that serves comprehension, not spectacle.

Accessible by default

Every text token meets WCAG contrast ratios. Reduced-motion preferences are honored. Focus states glow, not just outline.

Token-driven

No magic numbers. Every spacing, radius, shadow, and color traces back to a named token. Change one value, update everywhere.
