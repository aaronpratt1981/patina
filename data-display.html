Empty States
Placeholder content when a list or view has no data. Include an icon, a short message, and a primary action.

When to use / avoid
When to use
  • First-run experience or zero-result states
Avoid when
  • Loading state — use skeleton instead

  <div class="empty-icon" style="background:var(--accent-bg);">

Skeleton Loading
Animated placeholders that match the layout of real content. Reduces perceived load time and prevents layout shift.
Cards
List rows
When to use / avoid
When to use
  • Content is loading and layout is known
Avoid when
  • Layout is dynamic — use a simple spinner
