When to use / avoid
When to use
  • Multi-turn conversations with an AI assistant or copilot.
  • Support chat, onboarding wizards, or inline help powered by AI.
Avoid when
  • Displaying a single static AI response — use the Summary Card instead.
  • Simple yes/no prompts — use a confirmation dialog.

AI Options

Selectable modes that configure AI behavior. Radio-style selection with descriptions.

Creative
Exploratory, expressive responses with broader associations
Balanced
Well-rounded replies that blend accuracy with readability
Precise
Factual, concise answers with citations when available
Concise
Brief, to-the-point answers — minimal explanation
When to use / avoid
When to use
  • Letting users choose between AI modes, tones, or model configurations.
  • Settings panels where only one AI behavior can be active at a time.
Avoid when
  • Multiple options can be selected — use checkboxes or toggle switches instead.
  • More than 5 options — use a dropdown select.

When to use / avoid
When to use
  • Presenting AI-generated insights, reports, or analysis results.
  • Dashboards where users need quick AI-powered takeaways with confidence levels.
Avoid when
  • Content is user-authored — use a standard card instead.
  • The summary needs interactive editing — use the Chat Window for iterative refinement.
