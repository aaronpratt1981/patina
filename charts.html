Charts

Interactive Chart.js charts styled with Patina design tokens. Includes bar, stacked bar, line, area, doughnut, and radar types.

Bar Chart Stacked Bar Line Chart Area Chart Doughnut Chart Radar Chart
Bar Chart
Vertical bars comparing categorical data. Patina colors and rounded corners are applied automatically.
When to use / avoid
When to use
  • Comparing values across discrete categories
  • Showing ranked or ordered data
Avoid when
  • Data has a continuous x-axis (use line chart)
  • Comparing parts of a whole (use doughnut)
<div class="chart-container">
  <canvas id="my-bar"></canvas>
</div>
<script>
patinaChart('my-bar', 'bar', {
  labels: ['Mon','Tue','Wed','Thu','Fri'],
  datasets: [{ label: 'Revenue', data: [8400,11200,9800,14300,16100] }]
});
</script>
Stacked Bar Chart
Multiple datasets stacked within each bar to show composition and totals simultaneously.
When to use / avoid
When to use
  • Showing composition of totals across categories
  • Comparing both individual segments and overall values
Avoid when
  • Too many segments make middle values hard to compare
  • Individual segment comparison matters more than totals
<div class="chart-container">
  <canvas id="my-stacked"></canvas>
</div>
<script>
patinaChart('my-stacked', 'bar', {
  labels: ['Q1','Q2','Q3','Q4'],
  datasets: [
    { label: 'Product A', data: [12000,15000,18000,14000] },
    { label: 'Product B', data: [8000,9000,11000,13000] },
    { label: 'Product C', data: [5000,7000,6000,8000] }
  ],
  options: { scales: { x: { stacked: true }, y: { stacked: true } } }
});
</script>
Line Chart
Continuous data trends over time with smooth curves and point markers.
When to use / avoid
When to use
  • Showing trends over time
  • Comparing multiple series
Avoid when
  • Data is categorical with no inherent order
<div class="chart-container">
  <canvas id="my-line"></canvas>
</div>
<script>
patinaChart('my-line', 'line', {
  labels: ['Jan','Feb','Mar','Apr','May','Jun'],
  datasets: [
    { label: 'Users', data: [1200,1900,3000,5000,4200,6100] },
    { label: 'Sessions', data: [2400,3200,4100,5800,5100,7200] }
  ]
});
</script>
Area Chart
Line chart with a filled region beneath, useful for emphasizing volume or magnitude over time.
When to use / avoid
When to use
  • Emphasizing cumulative totals or volume
  • Showing change in magnitude over time
Avoid when
  • Multiple overlapping series obscure each other
<div class="chart-container">
  <canvas id="my-area"></canvas>
</div>
<script>
patinaChart('my-area', 'area', {
  labels: ['Jan','Feb','Mar','Apr','May','Jun'],
  datasets: [{ label: 'Pageviews', data: [4200,5100,4800,6300,7200,8100] }]
});
</script>
Doughnut Chart
Proportional data as slices of a ring. The center hole can display summary values or labels.
When to use / avoid
When to use
  • Showing parts of a whole (3-6 slices)
  • Proportional data that sums to 100%
Avoid when
  • More than 6 categories (hard to read)
  • Comparing values between groups (use bar)
<div class="chart-container-square">
  <canvas id="my-doughnut"></canvas>
</div>
<script>
patinaChart('my-doughnut', 'doughnut', {
  labels: ['Organic','Direct','Referral','Paid'],
  datasets: [{ data: [42, 25, 15, 18] }]
});
</script>
Radar Chart
Multi-axis radial chart for comparing features or skill profiles across dimensions.
When to use / avoid
When to use
  • Comparing profiles across 5-8 dimensions
  • Showing strengths and weaknesses
Avoid when
  • Too many axes make it unreadable (>8)
  • Exact values matter more than shape
<div class="chart-container">
  <canvas id="my-radar"></canvas>
</div>
<script>
patinaChart('my-radar', 'radar', {
  labels: ['Speed','Reliability','UX','Security','Scalability'],
  datasets: [
    { label: 'Product A', data: [85, 70, 90, 60, 75] },
    { label: 'Product B', data: [65, 85, 70, 80, 90] }
  ]
});
</script>
