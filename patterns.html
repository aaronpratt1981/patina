Error Section

Full-page error display for 404, 500, and maintenance states. Centered card with error code, message, and recovery actions.

404
Page not found
The page you're looking for doesn't exist or has been moved.
When to use / avoid
When to use
  • Dedicated error pages (404, 500, maintenance).
Avoid when
  • Inline errors within a page — use Alert or Empty State instead.

Usage Block

Paired do/dont guidance cards for documenting when to use or avoid a component.

When to use
  • Primary actions that complete a workflow.
  • Confirming destructive operations with clear labeling.
Avoid when
  • The action is passive or navigational — use a link.
  • Too many buttons compete for attention on one screen.
When to use / avoid
When to use
  • Component documentation pages needing do/don't guidance.
  • Design system guidelines with paired recommendations.
Avoid when
  • Simple inline help text is sufficient.
  • Guidance is better expressed as a single paragraph.
<div class="usage-block">
  <div class="usage-do">
    <div class="usage-label">When to use</div>
    <ul><li>Guideline text.</li></ul>
  </div>
  <div class="usage-dont">
    <div class="usage-label">Avoid when</div>
    <ul><li>Guideline text.</li></ul>
  </div>
</div>

Expand / Collapse

Collapsible content panel using a native <details> element. Warm header bar with label text and an Expand/Collapse pill indicator. No JavaScript required.

Collapsed
Section title
This content is revealed when the toggle is expanded. Any HTML can go here — text, lists, cards, or other components.
Expanded
Section title
This content is revealed when the toggle is expanded. Any HTML can go here — text, lists, cards, or other components.
When to use / avoid
When to use
  • Hiding optional or secondary content to reduce visual clutter.
  • FAQ sections, advanced settings, or lengthy details.
Avoid when
  • The content is critical and should always be visible.
  • Users need to compare information across multiple collapsed sections.
<details class="usage-toggle">
  <summary>Section title</summary>
  <div style="padding: 16px;">
    Collapsible content here.
  </div>
</details>
